Sistema de Gerenciamento Administrativo

Bem vindo ao sistema do clube XPTO, elaborado como trabalho de conclusão do curso Tecnologia em Sistemas de Computação do CEDERJ.

CEDERJ

Consórcio das universidades públicas do estado do Rio de Janeio, que provém cursos de graduação e de extensão na modalidade semi presencial

Java

Linguagem de programação orientada a objetos desenvolvida na década de 90,pela Sun Microsystems. Em 2008 adquirido pela empresa Oracle Corporation.

Spring Framework

É um framework open source para a plataforma Java, baseado nos padrões de projeto inversão de controle (IoC) e injeção de dependência.

JQuery

É uma biblioteca de funções JavaScript que interage com o HTML, desenvolvida para simplificar os scripts interpretados no navegador do cliente.

AJAX

Acrônimo (ingles) de Asynchronous Javascript and XML, faz uso metodológico de Javascript e XML, para tornar páginas Web mais interativas

Bootstrap

É um framework para desenvolvimento de componentes de interface e front-end para sites, baseado principalmente em CSS, objetivando um site amigável e responsivo.
